Android Studio 3.0 Canary版本上手体验

Posted by Csming on 2017-05-20
Csming
  • Home
  • Archives
  • About
  • Tags

Google2017年IO大会后推出Android Studio 3.0 Canary，金丝雀版本；正式将Kotlin作为Android的官方开发语言；
于是大会第二天我就迫不及待的想要尝试一下新的编译器了；

安装

没办法直接从2.x的稳定版直接升级到3.0；
于是我到官网直接下载了他的压缩包；
https://developer.android.com/studio/preview/install-preview.html#_1

然后解压，配置环境；

我这边是解压到了E:\Program Files\Android\Android Studio Preview这个文件夹里

新东西

  • 新增Profile or debug APK选项
    这个平时没怎么用就没在意；还是看网上的文章才注意到的；
  • Include Kotlin support
    在新建项目的时候，除了Include C++ support的勾选项外还增加了Include Kotlin support选项；
    而且，初始项目建完之后，我发现初始的MainActivity代码是用kotlin写的；

此外，新建项目时还增加了一个Android Things；

  • gradle下新增
    classpath “org.jetbrains.kotlin:kotlin-gradle-plugin:$kotlin_version”

gradle.build中改为：

看来谷歌要全面的支持kotlin语言了

其他内容

  • 图标更改了
    app的默认图标更换了

  • 版本迭代：

·即时应用支持

·项目建立速度改进

·Google Maven Repo更改

  • 优化改进：
    ·CPU分析器

·内存分析器

·网络分析器

·APK分析器

这部分内容感觉挺需要的；

  • 布局编辑器改进;XML字体以及可下载字体
    这个我挺好奇的……

好吧，体验结束；
因为其他一些内容暂时还玩不到；
像Google Play虚拟机，OpenGL ES 3.0虚拟机这些；

然后，测试与DeBug部分，因为我通常都直接用真机做测试，所以没怎么使用到；

感觉最重要的一个更新，还是google全面的去支持了Kotlin语言
这样一来，又要好好学习啦~

其他一些更新内容可以去看：http://www.tuicool.com/articles/juaURn3
